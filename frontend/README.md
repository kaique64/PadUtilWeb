# PadUtil - Leitura de inventário de placa

## Ferramentas utilizadas

- Vue.js
- Node.js
- Yarn
- Axios
- Vite.js

## Instale os pacotes

```sh
npm install
```

### Compile o código em ambiente de desenvolvimento

```sh
npm run dev
```

### Compile and minifique para o ambiente de produção

```sh
npm run build
```
